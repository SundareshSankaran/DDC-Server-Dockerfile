DDC server sample endpoints

Here are some sample endpoints which are provided with the DDC server. These are based on the samples available in the SAS Visual Analytics Third Party Visualizations GitHub repository.

Home Page

C3 Bar chart

As simple as they come, a bar chart from the C3.js library.

D3 Bar Chart

Another simple Bar Chart from the D3.js library.

D3 Circle Packing

Based the visualization available in the D3 library, this VA DDC sample uses a circular packing visualization which helps you view an heirarchical structure. Each node of the heirarchy is represented as a circle, with sub-nodes forming circles within the same.

D3 Network Diagram

Network diagrams are a great way to depict relationships and flow of information between different elements.
